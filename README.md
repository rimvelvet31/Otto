# Instructions to Run

### 1. Activate the virtual environment `.venv` by running:

**Windows**

```
.venv\Scripts\activate
```

**Mac & Linux**

```
source .venv/bin/activate
```

### 2. Run `main.py` in your terminal

**Windows**

```
python main.py
```

**Mac & Linux**

```
python3 main.py
```
